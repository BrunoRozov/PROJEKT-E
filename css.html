Mis on CSS?

CSS ehk Cascading Style Sheets on veebi disainimise keel, millega saab veebilehte dekoreerida ja muuta veebilehe stiili.

CSS ja HTML on nagu näpud ja klaviatuur. Ilma näputeta ei saa klaviatuuril kirjutada ja ilma klaviatuurita ei saa arvutis kirjutada. Üks ei tööta kui teine ei ole olemas.

Kui HTML on veebilehe struktuur, siis CSS on selle stiil ja paigutus.
